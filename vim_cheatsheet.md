# Vim Cheatsheet

## Enter Insert Mode
*(Uppercase letter alternative: `SHIFT + lowercase letter`)*

- `i` – Start editing left from cursor  
- `I` – Start editing at the start of the line  
- `a` – Start editing left from cursor  
- `A` – Start editing at the end of the line  

## Exit Insert Mode

- `<Esc>` – Enter Normal mode  

## Save the Current Buffer (File)

- `:w`

## Quit Vim

- `:q`

## Save and Quit

- `ZZ` – (or `Shift + Z + Z`, same thing)  
- `:wq`

## Exit Recording Mode  
*(Displays `@recording` at the bottom when accidentally entered)*

- `q`
